------------------------------------------------------------
Installing pyNBS (REQUIRED)
------------------------------------------------------------

The pyNBS package is included in:

  DNA_pyNBS-SBI/

To install:

  cd DNA_pyNBS-SBI
  pip3 install .

This installs:
  • pyNBS.data_import_tools
  • pyNBS.network_propagation
  • all helper modules

After installation you may test:

  python3 -c "import pyNBS; print('pyNBS OK')"
